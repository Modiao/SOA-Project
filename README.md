# SOA-Project
SOA Project Front with Vue js &amp; Back with Spring Boot
## Installation

### Clone repo

``` bash
# clone the repo
$ git clone https://github.com/vincediegane/SOA-Project.git my-project

# go into app's directory
$ cd my-project

# install app's dependencies
$ npm install
```

## Usage

``` bash
# serve with hot reload at localhost:3000.
$ npm run serve

# build for production with minification
$ npm run build
```

## What's included
